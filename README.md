## Install

###React app

```sh
$ git clone https://github.com/EpitechWebAcademiePromo2022
$ cd react-chat
$ yarn install || npm install
$ yarn start || npm run start
```

###Slim api

```sh
$ git clone https://github.com/EpitechWebAcademiePromo2022
$ cd slim
$ composer install
$ composer start
```

###Node api

```sh
$ git clone https://github.com/EpitechWebAcademiePromo2022
$ cd api_express
$ yarn install || npm install
$ yarn start || npm run start
```

###Connector api

```sh
$ git clone https://github.com/EpitechWebAcademiePromo2022
$ cd api_connector
$ composer install
$ composer start
```
